# Sherbrooke East Project — Renovations (PWA)

Mobile-first room-by-room renovation tracker with photos, notes, and checklists.

## Quickstart
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to Vercel
1. Push this folder to a new GitHub repo (e.g., `sherbrooke-east-renovation-app`).
2. Go to https://vercel.com/import/git and select your repo.
3. Framework: **Vite / React**, Build: `npm run build`, Output: `dist`, Install: `npm install`.
4. Deploy ✅

## PWA
- Installable on iOS/Android via Add to Home Screen
- Offline shell via `public/sw.js`
- Icons + manifest included
